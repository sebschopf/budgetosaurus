Glossaire des Concepts

Comprenez les termes clés de votre application de gestion budgétaire.

Compte (Account)

Un Compte représente un lieu physique ou logique où votre argent est détenu. Il peut s'agir d'un compte bancaire courant, d'un compte d'épargne, d'une carte de crédit, ou même d'une caisse d'espèces. Chaque compte a un solde réel qui est affecté par les transactions.

Analogie : C'est le tiroir ou le portefeuille où se trouve l'argent.

Catégorie (Category)

Une Catégorie est une étiquette descriptive qui classe la nature d'une transaction. Elle répond à la question "À quoi sert cet argent ?" (pour une dépense) ou "D'où vient cet argent ?" (pour un revenu). Les catégories peuvent être hiérarchiques (par exemple, "Dépenses Fixes" > "Loyer").

Les catégories sont au cœur de votre gestion financière, car elles peuvent être associées à des Fonds, faire partie d'un Budget, ou être liées à un Objectif d'Épargne. Leur statut est indiqué par des tags visuels (flags) à côté de leur nom dans les menus déroulants.

Analogie : C'est l'étiquette sur le tiroir qui décrit ce qu'il contient (ex: "factures", "nourriture").

Transaction

Une Transaction est un enregistrement d'un mouvement d'argent. Chaque transaction est liée à une date, une description, un montant, un compte et une catégorie. Elle peut être de différents types :

  • Revenu (IN) : Représente de l'argent entrant sur votre compte. C'est utilisé pour votre salaire, des remboursements reçus, des intérêts, etc.
  • Dépense (OUT) : Représente de l'argent sortant de votre compte. C'est utilisé pour vos achats, factures, retraits, etc.
  • Transfert (TRF) : Représente un mouvement d'argent entre deux de vos propres comptes (par exemple, d'un compte courant à un compte d'épargne). Ce type est important pour éviter de compter deux fois l'argent dans vos totaux de revenus/dépenses, car il ne s'agit pas d'un vrai revenu ou d'une vraie dépense, mais d'un simple déplacement.

Il est crucial de choisir le bon type de transaction pour que vos récapitulatifs financiers soient précis.

Analogie : C'est une ligne dans votre relevé bancaire ou votre carnet de comptes.

Budget (Planification)

Un Budget est un montant que vous prévoyez de ne pas dépasser pour une catégorie donnée sur une période spécifique (par exemple, mensuelle ou annuelle). C'est une cible ou une limite que vous vous fixez pour vos dépenses. Une catégorie est "budgétisée" si un budget lui est associé, et est identifiée par un tag visuel de couleur **Bleu** dans les menus déroulants.

Analogie : C'est la limite que vous vous fixez pour la quantité d'argent que vous allez dépenser pour le "carburant" ce mois-ci.

Fonds (Provision)

Un Fonds représente une "enveloppe" budgétaire virtuelle pour une catégorie spécifique, dont le solde est cumulatif et automatiquement ajusté par les transactions qui lui sont associées.

Ce concept est utile si vous souhaitez mettre de côté de l'argent pour des dépenses futures ou gérer un solde roulant pour certains objectifs (par exemple, "Fonds Vacances", "Fonds Impôts"). Une catégorie est dite "gérant un fonds" si elle est configurée pour ce suivi de provision, et est identifiée par un tag visuel de couleur **Verte** dans les menus déroulants.

Les transactions d'une catégorie qui ne gère pas de fonds impacteront uniquement le "Récapitulatif Mensuel par Catégorie" et le solde de votre compte, sans ajuster un fonds dédié.

Analogie : C'est une tirelire dédiée à un objectif spécifique (ex: "Vacances"). Vous y ajoutez de l'argent (revenus, allocations) et en retirez (dépenses). Le solde de la tirelire vous indique combien il reste pour cet objectif.

Objectif d'Épargne (SavingGoal)

Un Objectif d'Épargne est un montant spécifique que vous voulez accumuler d'ici une certaine date pour un but précis (par exemple, acheter une voiture, payer un acompte pour une maison). Il s'agit d'un objectif à long terme pour une dépense importante et souvent non récurrente.

Une catégorie est "liée à un objectif" si elle est associée à un objectif d'épargne actif, et est identifiée par un tag visuel de couleur **Orange/Jaune** dans les menus déroulants.

Analogie : C'est la somme totale que vous voulez atteindre pour un achat important, comme le prix d'une voiture, et vous y contribuez de diverses manières.

Tag / Flag (Indicateur Visuel de Catégorie)

Les Tags / Flags sont de petits badges colorés qui apparaissent à côté du nom des catégories dans les menus déroulants. Ils servent d'indicateurs visuels rapides pour vous informer de la nature ou de l'impact particulier de la catégorie sélectionnée :

  • Fonds (Vert) : Indique que la catégorie gère une provision ou une "enveloppe" budgétaire cumulative.
  • Budget (Bleu) : Indique que la catégorie est associée à un budget planifié.
  • Objectif (Orange/Jaune) : Indique que la catégorie est liée à un objectif d'épargne spécifique.

Une catégorie peut avoir plusieurs de ces tags si elle répond à plusieurs critères (par exemple, une catégorie peut être à la fois un "Fonds" et avoir un "Budget" si vous planifiez d'y verser un certain montant chaque mois).

Analogie : Ce sont comme de petits autocollants de couleur sur vos étiquettes de tiroir qui vous donnent des informations supplémentaires d'un coup d'œil.
